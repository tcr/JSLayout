  • #include for JavaScript

    Extending the capabilities of the <script> tag.

    Inspired by a post about degrading script tags, I decided to add an #include directive to JavaScript, in the style of traditional C preprocessors.

    While it may be absurd, it still beats writing out all those <script src="..." /> tags...

    Originally posted on August 29, 2008.

  • INIFile.php

    A small (4kb) PHP class for reading/writing INI files.

    INIFile goes beyond the default parse_ini_file functionality, allowing your application to read as well as write configuration files.

    This is, of course, a light-weight alternative for configuration than more code-heavy solutions, such as XML. (Not to say XML is bad for configuration—but PHP is not Java!)

    Originally posted on July 2, 2008.

  • WebTV Image Viewer

    A script for viewing full-size images on a WebTV or MSNTV box.

    I've always adored the WebTV for it's ancient techonology and exotic way of interacting with the web. Imagine the enticement of the challenge I faced when I found that it genuinely had no way of scrolling horizontally, only vertically. Thrilling!

    In an exercise of masochism and ancient proprietary code references, I developed a way to intuitively view images which extended beyond the viewport borders, using the arrow keys.

    And no, I don't know why I ever spent so much time on this either. View the source if you want a good laugh. Or cry. (Or even take it for a spin!)

    Originally posted on July 31, 2006.

  • BloopJS

    A JavaScript interpreter for the BlooP and FlooP programming languages.

    My love for esoteric programming languages met my fancy for the name "bloope" when I decided to port an interpreter for this strange lanugage from Perl to JavaScript.

    Regardless if you've heard of BlooP or not, I would definitely recommend reading Douglas Hofstadter's Gödel, Escher, Bach for anyone with a mind and a curiosity as to how it got there.

    Credit goes to John Cowan for writing the original interpreter.

    Originally posted on October 20, 2004.
